# LLM Building & Training Primer

[![License: MIT](https://img.shields.io/github/license/larkindom/LLMTrainingLecture)](LICENSE)
[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen)](https://larkindom.github.io/LLMTrainingLecture/)

Interactive, single-page notes on **"Pretraining, Post-Training, and the Parts Nobody Puts on Slides"** —
built from the transcript of Yann Dubois's guest lecture in Stanford's CS229: Machine Learning (Summer 2024).

The lecture covers how ChatGPT-like models are actually built: the next-token training objective and
tokenization, evaluating a base model, the data pipeline, scaling laws, the economics of a frontier
training run, and post-training (SFT, RLHF, DPO) — plus how to evaluate an aligned assistant and the
systems tricks that make training affordable.

## View it

**[Open the live primer](https://larkindom.github.io/LLMTrainingLecture/)**, or open
[`index.html`](./index.html) directly in a browser — it's a single self-contained file, no
build step required. Click through the 12 topics in the left-hand table of contents, or use
the Next/Previous buttons and the ← / → arrow keys. Two topics include live interactive
demos: next-token sampling and byte-pair-encoding tokenization.
